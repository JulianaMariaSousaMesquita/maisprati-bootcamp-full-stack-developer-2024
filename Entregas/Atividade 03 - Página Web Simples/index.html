Lógica de Programação

A lógica de programação é a base de qualquer linguagem de programação. Ela envolve a criação de algoritmos para resolver problemas de maneira eficiente. Entender lógica é o primeiro passo para se tornar um programador.

Condicionais

Condicionais são estruturas que permitem a execução de diferentes blocos de código com base em uma condição. Exemplos incluem o famoso if, else e switch. 

if (idade > 18) {
    console.log('Maior de idade');
} else {
    console.log('Menor de idade');
}

Tabela Verdade

A tabela verdade é uma ferramenta usada para entender operações lógicas, como AND, OR e NOT, e como elas se comportam com diferentes combinações de valores booleanos (verdadeiro ou falso).

A B A AND B A OR B NOT A

Verdadeiro Falso Falso Verdadeiro Falso
Falso Verdadeiro Falso Verdadeiro Verdadeiro
Falso Falso Falso Falso Verdadeiro
